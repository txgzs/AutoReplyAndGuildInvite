# 使用说明：
## 魔兽世界时光服插件：自动回复密语关键字+公会邀请
## 插件文件夹AutoReplyAndGuildInvite解压缩到World of Warcraft\_classic_titan_\Interface\AddOns
## 调用插件设置命令为/argi
## 主要功能为自动回复关键字+公会邀请
## 目前无法自动邀请密语者加公会，需要手动点击屏幕中间弹出的按钮来邀请公会。
## 新增快捷键按“]”来执行邀请按钮
