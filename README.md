# 使用说明：
## 魔兽世界时光服插件：自动回复密语关键字+公会邀请
## 1. 插件文件夹AutoReplyAndGuildInvite解压缩到World of Warcraft\_classic_titan_\Interface\AddOns
## 2. 调用插件设置命令为/argi
## 3. 主要功能为自动回复关键字+公会邀请
首次收到密语会发给对方一个提示信息，插件会记录发送密语的角色名到文件中避免无限发送提示信息，可以清空此文件记录的角色名。
## 4. 目前无法自动邀请密语者加公会，需要手动点击屏幕中间弹出的按钮来邀请公会。
## 5. 新增快捷键按“]”来执行邀请按钮
## 6. 可以通过保存复制配置文件来达到跨设备使用，而不用重新设置。World of Warcraft\_classic_titan_\WTF\Account\你的账号\SavedVariables\AutoReplyAndGuildInvite.lua
